🔄 Limpar Cache do Navegador

⚠️ IMPORTANTE: O erro de datas inválidas está acontecendo porque seu navegador está usando código JavaScript antigo em cache.

Opção 1: Limpar Cache Automaticamente

Clique no botão abaixo para tentar limpar o cache automaticamente:

Opção 2: Limpar Cache Manualmente (RECOMENDADO)

Windows/Linux:

  1. Pressione Ctrl + Shift + Delete
  2. Selecione "Imagens e arquivos em cache"
  3. Clique em "Limpar dados"
  4. Pressione Ctrl + Shift + R para recarregar

Mac:

  1. Pressione Cmd + Shift + Delete
  2. Selecione "Imagens e arquivos em cache"
  3. Clique em "Limpar dados"
  4. Pressione Cmd + Shift + R para recarregar

Opção 3: Usar Aba Anônima

Abra uma nova aba anônima/privada:

  • Chrome/Edge: Ctrl + Shift + N
  • Firefox: Ctrl + Shift + P

Depois acesse: http://localhost:3000

Opção 4: DevTools (Para Desenvolvedores)

  1. Pressione F12 para abrir DevTools
  2. Clique com botão direito no ícone de reload
  3. Selecione "Empty Cache and Hard Reload"
✅ Cache limpo! Redirecionando para a aplicação...
📝 Nota: Após limpar o cache, o erro RangeError: Invalid time value deve desaparecer completamente.
