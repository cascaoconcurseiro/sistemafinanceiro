# Logos dos Bancos

Esta pasta contém as logos SVG dos bancos brasileiros.

## Como adicionar as logos:

1. Baixe o repositório: https://github.com/Tgentil/Bancos-em-SVG
2. Copie os arquivos SVG da pasta `bancos/` para esta pasta `public/banks/`
3. Os arquivos devem seguir o padrão: `[codigo]-[nome].svg`
