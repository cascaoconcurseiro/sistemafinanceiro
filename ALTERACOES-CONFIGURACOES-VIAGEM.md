# ✅ Alterações nas Configurações da Viagem

## 🎯 O Que Foi Feito

### 1. ✅ Removida Seção de "Participantes"
- **Antes**: Havia uma seção completa para adicionar/remover participantes
- **Depois**: Seção removida das configurações
- **Motivo**: Participantes devem ser gerenciados na aba "Família", pois é lá que são vinculadas as despesas compartilhadas

### 2. ✅ Edição na Visão Geral
- A aba "Visão Geral" já permite visualizar todos os dados da viagem
- Para editar, o usuário deve ir na aba "Configurações"
- Lá ele pode editar:
  - Nome da viagem
  - Destino
  - Datas (início e fim)
  - Orçamento
  - Moeda
  - Descrição

## 📝 Fluxo Correto

### Para Adicionar Participantes
1. Vá para o menu **"Família"**
2. Adicione os membros da família
3. Ao criar despesas compartilhadas na viagem, selecione os membros

### Para Editar Dados da Viagem
1. Abra a viagem
2. Vá para a aba **"Configurações"**
3. Clique em **"Editar"**
4. Altere os dados desejados
5. Clique em **"Salvar"**
