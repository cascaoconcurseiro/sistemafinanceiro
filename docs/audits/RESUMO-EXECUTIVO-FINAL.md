## 🚀 COMO USAR

### Para Usuários

1. **Acesse o sistema:**
   ```
   http://localhost:3000
   ```

2. **Crie sua primeira transação:**
   - Pressione `Ctrl+N`
   - Ou clique no botão "+"
   - Ou use `Ctrl+K` → "Nova Transação"

3. **Veja insights automáticos:**
   - Dashboard → Seção "Insights"
   - Sistema analisa e sugere ações

4. **Use atalhos para produtividade:**
   - `Ctrl+K` → Command Palette
   - `Ctrl+N` → Nova Transação
   - `Ctrl+F` → Buscar

### Para Desenvolvedores

1. **Executar testes:**
   ```bash
   npm test
   npm run test:coverage
   ```

2. **Executar auditoria:**
   ```bash
   # Via navegador
   http://localhost:3000/audit
   
   # Via script
   npm run audit
   ```

3. **Monitorar performance:**
   ```typescript
   import { performanceMonitor } from '@/lib/performance/monitor'
   
   const result = await performanceMonitor.measureAsync('operation', async () => {
     return await doSomething()
   })
   ```

4. **Usar categorização automática:**
   ```typescript
   import { smartSuggestCategory } from '@/lib/ml/auto-categorization'
   
   const suggestion = smartSuggestCategory('Uber para casa')
   // { category: 'transporte', confidence: 0.95 }
   ```
