# 🚀 COMO INICIAR O SERVIDOR

**Problema**: O comando `npm` não está disponível no PATH do sistema.

---

## ✅ SOLUÇÃO

### Opção 1: Usar Terminal Externo (Recomendado)

1. **Abra um terminal externo** (CMD, PowerShell ou Terminal do Windows)

2. **Navegue até o projeto**:
   ```bash
   cd "Não apagar\SuaGrana-Clean"
   ```

3. **Execute o servidor**:
   ```bash
   npm run dev
   ```

4. **Acesse no navegador**:
   ```
   http://localhost:3000
   ```

### Opção 2: Adicionar Node.js ao PATH

1. **Encontre o caminho do Node.js**:
   - Geralmente: `C:\Program Files\nodejs\`
   - Ou: `C:\Users\[SEU_USUARIO]\AppData\Roaming\npm`

2. **Adicione ao PATH**:
   - Abra "Variáveis de Ambiente"
   - Edite a variável PATH
   - Adicione o caminho do Node.js
   - Reinicie o terminal

3. **Teste**:
   ```bash
   node --version
   npm --version
   ```
